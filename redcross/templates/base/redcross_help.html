  • What is the Red Cross Apps website?
    • This is a website that initially contains just the RIMS database app. More apps can be added over time as the need arises.
  • The home page says that the site is hosted by UlsterCorps.org. What is that?
    • UlsterCorps.org is a countywide resource, in Ulster County NY, dedicated to fostering a culture of volunteerism, collaborative work and community service.
  • What is RIMS?
    • RIMS stands for Red Cross Inventory Management System. It was initially developed by Eugene Ouderkirk using Microsoft Access. Later Rob Groves converted the Access database into this web app.
  • How do I add or subtract quantity from existing inventory at a site?
    1. Select the Sites menu item
    2. Page through the list of sites and click on the desired site. This will take you to the Site Details page. On the Site Details page you will see site information on the left and site inventory on the right.
    3. Enter a new quantity for each inventory item you want to edit.
    4. When finished, hit "Save Changes" in the upper right corner of the inventory list.
  • How do I add a new product into a site's inventory?
    1. Select the Sites menu item
    2. Page through the list of sites and click on the desired site. This will take you to the Site Details page. On the Site Details page you will see site information on the left and site inventory on the right.
    3. Click on the "Add New Inventory" button in the upper left corner of the site inventory list. This will take you to the Add New Inventory page, where you will once again see the site information on the left, but this time you will see a list of products that can be added on the right.
    4. Select the product(s) you want to add by clicking in the checkboxes.
    5. When you have all the new products selected, hit "Add Products" in the upper right corner of the product list. This will take you to a page where you can set the new quantity of each of the new products. If you happened to select a product that already existed at the site, you will see the current inventory quantity at the site for that product.
    6. Set the quantity for each product.
    7. When finished, hit "Save Inventory" to save your changes and return to the Site Details page, or hit "Cancel" to return to the Site Details page without adding any products.

  • How do I generate a site by site inventory detail report?
    1. Select the Reports menu item.
    2. Today's date will be entered for you in the "start" and "end" report dates fields.
    3. If you want to take a snapshot of the inventory at a specific date in the past, select a different "end" date. The "start" date field is not currently used. This will be used to generate inventory change reports in a future software revision.
    4. Select the "print view" button under the "Site Inventory" heading. The "xls" report button is not currently enabled. This will be used to generate excel formatted reports in a future software revision.
    5. To print the report, hit the "print" button in the upper left corner of the report frame.

    1. Select the Reports menu item.
    2. Ignore the date fields. Site detail reports are not filtered by date.
    3. Select the "print view" button under the "Site Detail" heading.
    4. To print the report, hit the "print" button in the upper left corner of the report frame.
  • What does it mean when there is a red box around a site or product?
    • This means that the site or product has some missing information in the database. This is usually due to the fact that the information was imported in bulk and the spreadsheet used was incomplete. This red box can in general be ignored. If it is desired to edit the information for a site or product that has missing information, you will have to enter the missing information beofre the database will allow you to save.
  • How do I edit a site's information (possibly to make that irritating red box go away!)?
    1. Select the Sites menu item
    2. Page through the list of sites and click on the desired site. This will take you to the Site Details page. On the Site Details page you will see site information on the left and site inventory on the right.
    3. Enter new site information in the left hand box.
    4. When finished, hit "Save Site" in the upper left corner of the site box.
    5. If required information is missing, you will be prompted to enter it before saving. Simply entering "none" in the missing information field is sufficient to allow the site to be saved. It is preferable to enter the correct information if it is available.
  • How do I add a new site?
    1. Select the Sites menu item
    2. Click on the "Add" button in the upper right corner of the sites list.
    3. Fill in the information and hit "Save Site" in the upper left corner of the site box.
    4. If required information is missing, you will be prompted to enter it before saving. Simply entering "none" in the missing information field is sufficient to allow the site to be saved. It is preferable to enter the correct information if it is available.
  • How do I delete a site completely from the database?
    1. Select the Sites menu item
    2. Page through the list of sites and click in the checkbox of the desired site(s).
    3. Hit "Delete" in the upper right corner of the sites list. This will take you to a confirmation page. Just confirm the deletion by hitting "Delete Site", or "Cancel". You will be returned to the Site List page.
      • Note:
      • Deleting a product will delete all existing inventory associated with that product at all sites.

  • How do I import Product, Site and/or Inventory Information in bulk?
    1. Select the Import/Export menu item
    2. Select a file from which to import using the File: Browse button.
    3. Depending on the type of information in the file (Site, Product Information, Inventory), hit the appropriate "Import" button.
      • Note:
      • If you want to completely replace the infomation in the database, hit the appropriate "Delete" button before importing.
      • Importing Site information will overwrite existing site information.
      • Importing Product information will overwrite existing product information.
      • Importing Inventory information will add new inventory change entries into the site inventory histories.
  • How do I completely delete all Inventory information in the database so I can start over?
    1. Select the Import/Export menu item
    2. Hit the "Delete" button under the Inventory heading.
      • Note:
      • All inventory can deleted without disturbing current Site and Product information in the database.
  • How do I completely delete all Site and/or Product information in the database so I can start over?
    1. Select the Import/Export menu item
    2. Hit the "Delete" button under either the Products or Sites heading.
      • Note:
      • Deleting all Products will also delete all inventory, since they are related in the database.
      • Deleting all Sites will also delete all inventory, since they are related in the database.
  • How do I export Product and/or Site information to an Excel file?
    1. Select the Import/Export menu item
    2. Hit the "All" button under the "Exports" heading for either Products or Sites.
      • Note:
      • Youlwil be prompted to either open the spreadsheet or to download or save it.
      • If you download/save the Excel file, it will show up on your computer, most likely in the Downloads directory.
      • The resulting Excel file can be used as a template for importing information back into the RIMS database.
      • One possible scenario would entail downloading either the Sites or Products information to Excel and then making changes to the information for one or more Products or Sites. The resulting file could be imported back into the RIMS database to update old information.
  • How do I export Inventory information to an Excel file?
    1. Select the Import/Export menu item
    2. Hit either the "All" or "Current" button under the "Exports" heading for Inventory.
    3. "All" will export all inventory including a history of all changes made to inventory items at each site, including inventory items that have been deleted.
    4. "Current" will export the current inventory for all sites.
      • Note:
      • You will be prompted to either open the spreadsheet or to download or save it.
      • If you download/save the Excel file, it will show up on your computer, most likely in the Downloads directory.
      • The resulting Excel file can be used as a template for importing inventory information back into the RIMS database.
      • One possible scenario would entail downloading "Current" Inventory information to Excel and then making changes to the information for one or more Products or Sites. The resulting file could be imported back into the RIMS database to enter new inventory information. Inventory information that did not need to be updated would be deleted from the spreadsheet before re-importing.
  • How do I create an Excel backup file of the entire database?
    1. Select the Import/Export menu item
    2. Hit the "Backup" button under the "Backup/Restore" heading.
    3. This creates a complete backup of the current state of the RIMS database, including all historical changes"
      • Note:
      • You will be prompted to either open the spreadsheet or to download or save it.
      • If you download/save the Excel file, it will show up on your computer, most likely in the Downloads directory.
      • The resulting Excel file can be used to do a restore of the RIMS database in the event of a problem.
  • How do I restore the RIMS database froma previously generated Excel backup file?
    1. Select the Import/Export menu item
    2. Hit the "Restore" button under the "Backup/Restore" heading.
    3. Select a file from which to restore using the File: Browse button.
      • Note:
      • You will be prompted to confirm the restore operation.
      • This will first delete all information in the database. Be sure you know what you are doing!
      • When the restore is complete, the database will exactly match the information in the backup Excel file.
